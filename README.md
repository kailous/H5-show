# H5-show
一个简易的H5展示开发框架

## 目录结构

``` sh
project/
│
├── components/ # 组件目录
│ ├── form-component.html # 表单组件
│ └── head.html # HTML 头部组件
│
├── pages/ # 页面目录
│ ├── 01.html # 第一页
│ ├── 02.html # 第二页
│ └── ... # 其他页
│
├── public/ # 公共资源
│ ├── animate/ # 动画库
│ ├── audio/ # 音频文件
│ ├── fonts/ # 字体库
│ ├── css/ # 样式文件
│ ├── js/ # JavaScript 文件
│ └── img/ # 图片文件
│
├── index.html # 入口 HTML 文件
├── index.js # 主要的 JavaScript 文件
└── package.json # 项目配置文件
```
